Comment ça marche ?

1️⃣ Modifie ton profil, tes projets, ta galerie via les menus à gauche.

2️⃣ Sauvegarde avec le bouton vert — les données sont stockées dans ton navigateur.

3️⃣ Visualise avec le bouton "Voir le site" — le portfolio se met à jour automatiquement.

4️⃣ Pour mettre en ligne : glisse les deux fichiers HTML sur GitHub Pages ou Netlify.

Option 2 — GitHub (le plus propre) :
Crée un repo → Upload le PDF → Clique dessus → "Raw" → Copie l'URL
